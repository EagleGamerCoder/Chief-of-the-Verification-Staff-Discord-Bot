# Chief-of-the-Verification-Staff-Discord-Bot

Current Version: V 0.3 

Made using: Python 3.14.3 

---------------

This file is a discord bot for the **Calderian Army** discord servers, [Link to main server](https://discord.com/channels/1484960694431383663/1487920774713966834).

It verifies and updates users and their roles in the servers based upon their roblox group rank.

It is deployed using render as a 'WEB SERVICE' and then monitored using uptimerobot.
