# Chief-of-the-Verification-Staff-Discord-Bot
Current Version: V 0.3 \n
Made using: Python 3.14.3 \n\n
This file is a discord bot for the Calderian Army discord servers.\n
It verifies and updates users and their roles in the servers based upon their roblox group rank.\n
It is deployed using render as a 'WEB SERVICE' and then monitored using uptimerobot\n
